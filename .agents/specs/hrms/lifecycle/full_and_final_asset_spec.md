Module Name: Full and Final Asset
Module: HRMS / HR
Type: Child Table (of Full and Final Statement)
DocStatus: No

Description
A single asset row (e.g., laptop, ID card, access card) listed on the Full and Final Statement that the employee must return.

Parent
Full and Final Statement (field: assets_to_be_returned)

Dependencies
None

Schema Fields

asset_category: Data, Optional.
date: Date, Optional.
status: Select (Pending Return, Returned), Optional.
